Introducción

¿Qué es Python?

Python es un lenguaje de programación creado por Guido van Rossum a finales de los ochenta cuya filosofía hace hincapié en la legibilidad de su código. Su nombre está inspirado por la afición de su creador en el grupo de cómicos ingleses Monty Python.

Se trata de un lenguaje interpretado, miltiplataforma, con tipado dinámico, fuertemente tipado y orientado a objetos.

Es usado para diferentes propósitos

  • Desarrollo web (del lado del servidor)
  • Inteligencia artificial
  • Ciencia de datos
  • Desarrollo de software
  • Crear videojuegos
  • Scripting

¿Qué puede hacer Python?

  • Python se puede utilizar en un servidor para crear aplicaciones web
  • Python se puede utilizar junto con el software para crear flujos de trabajo
  • Python puede conectarse a sistemas de bases de datos. También puede leer y modificar archivos
  • Python se puede utilizar para manejar grandes cantidades de datos y realizar matemáticas complejas
  • Python se puede utilizar para la creación rápida de prototipos o para el desarrollo de software listo para producción

¿Por qué Python?

  • Python es un lenguaje multiplataforma, lo que significa que funciona en diferentes plataformas (Windows, Mac, Linux, Raspberry Pi, etc.)
  • Python tiene una sintaxis simple similar al idioma inglés
  • Python tiene una sintaxis que permite a los desarrolladores escribir programas con menos líneas que otros lenguajes de programación
  • Python se ejecuta en un sistema de interpretación, lo que significa que el código se puede ejecutar tan pronto como se escribe. Esto significa que la creación de prototipos puede ser muy rápida
  • Python puede tratarse de forma procedimental, orientada a objetos o funcional.
  • Python tiene una comunidad que incluye un grupo de personas increíblemente diverso y acogedor. Tener una comunidad bien conectada y de apoyo es fundamental para ayudar a resolver problemas, y la comunidad de Python apoya plenamente a personas que están aprendiendo Python como su primer lenguaje de programación.

Bueno saber

  • La versión principal más reciente de Python es Python 3, que se usará en esta documentación. Sin embargo, Python 2, aunque no se actualiza con nada más que actualizaciones de seguridad, sigue siendo bastante popular.
  • En esta documentación, Python se escribirá en la terminal y en un editor de texto. Puede utilizar un editor de código como Visual Studio Code, Sublime Text, Atom, o también en un entorno de desarrollo integrado, como Thonny, Pycharm, Netbeans, Eclipse, que son particularmente útiles cuando se administran colecciones más grandes de archivos Python. Además, también puede utilizar la terminal de Python o incluso el editor de texto si lo desea.

Sintaxis de Python en comparación con otros lenguajes de programación

  • Python fue diseñado para facilitar la lectura y tiene algunas similitudes con el idioma inglés con influencia de las matemáticas.
  • Python usa nuevas líneas para completar un comando, a diferencia de otros lenguajes de programación que a menudo usan punto y coma o paréntesis.
  • Python se basa en la sangría, usando espacios en blanco, para definir el alcance; como el alcance de los bucles, funciones y clases. Otros lenguajes de programación a menudo usan corchetes para este propósito.

Python es un gran lenguaje para aprender, ¡así que comencemos!

Empezar

Primero, verificaremos si hay una versión reciente de Python instalada en su sistema y la instalara si no lo está.

Python en Windows

Windows no siempre viene con Python, por lo que probablemente necesitara instalarlo.

Python en macOS

Python ya está instalado en la mayoría de los sistemas macOS, pero lo más probable es que sea una versión desactualizada de la que no querrá aprender.

Python en Linux

Los sistemas Linux están diseñados para la programación, por lo que Python ya está instalado en la mayoría de las computadoras Linux.

Instalar Python

Primero, verifique si Python está instalado en su sistema. Abra una ventana de terminal en Windows ingresando command (o terminal) en el menú Inicio y presione enter en la aplicación que aparece o presione la combinación de teclas Windows + r, y en la ventana que aparece escriba cmd y presione enter; en Mac abra una ventana de terminal yendo a Applications > Utilities > Terminal, también puede presionar la tecla Command, y escribir terminal y luego presionar Intro; en Linux abra una ventana de terminal ejecutando la aplicación Terminal en su sistema (en Ubuntu, puede presionar ctrl-alt-T).

En la ventana de terminal, ingrese python en minúsculas en Windows y Mac (en Linux escriba python3). Si obtiene un indicador de Python (>>>) como respuesta, Python está instalado en su sistema. Si ve un mensaje de error que indica que Python no es un comando reconocido, Python no está instalado.

En ese caso, o si ve una versión de Python anterior a Python 3, debe descargar un instalador de Python desde la página oficial. Vaya a la página oficial de Python y coloque el cursor sobre Downloads. Debería ver un botón para descargar la última versión de Python. Haz clic en el botón, que debería comenzar a descargar automáticamente el instalador correcto para su sistema.

Una vez que haya descargado el archivo, ejecute el instalador. Asegúrese de seleccionar la opción Add Python to PATH, lo que facilitará la configuración correcta de su sistema, y presione en Install now.

Iniciar con Python (Hola, Mundo!)

Una vez instalado Python en su sistema, abra la terminal de Python y escriba su primer programa.

Ingrese la siguiente línea en su sesión de Python y asegúrese de ver el resultado: Hello, World! 


                    

                        >>> 
                        print
                        (
                        "
                        Hello, World!
                        "
                        )
                    

                    

                        Hello, World!
                    

                    

                        >>>

Perfecto, ha instado Python correctamente en su sistema.

La línea de comandos de Python

Para probar una pequeña cantidad de código en Python, a veces es más rápido y fácil escribirlo directamente en la terminal de Python y no escribir el código en un archivo. Esto es posible porque Python se puede ejecutar como una línea de comandos.

Siempre que haya terminado de trabajar en la línea de comando de Python, simplemente puede escribir lo siguiente para salir de la interfaz de línea de comando de Python: exit() o ctrl + z (en Windows)

Crear archivos Python

Python es un lenguaje de programación interpretado, esto significa que, como desarrollador, escribe archivos Python (.py) en un editor de texto y luego coloca esos archivos en el intérprete de Python para que se ejecuten o también puede ejecutar el archivo directamente desde un editor de código o IDE que esté utilizando.

La forma de ejecutar un archivo python es así en la línea de comando: 


                    

                        C:\Users\Your Name>python helloworld.py

Donde helloworld.py es el nombre de su archivo Python.

Escribamos nuestro primer archivo Python, llamado helloworld.py, que se puede hacer en cualquier editor de texto.

Primero cree un archivo desde el editor de texto que esté utilizando llamado: helloworld.py, e ingrese la siguiente línea de código (asegúrese de que la extensión del archivo que creo sea .py: 


                    

                        print("Hello, World!")

Primero cree un archivo desde el editor de texto que esté utilizando llamado: helloworld.py, e ingrese la siguiente línea de código (asegúrese de que la extensión del archivo que creo sea .py: 


                    

                        C:\Users\Your Name>python helloworld.py

La salida debería verse así: 


                    

                        Hello, World!

Felicitaciones, ha escrito y ejecutado su primer programa Python.

Sangría de Python

La sangría se refiere a los espacios en blanco al comienzo de una línea de código.

Mientras que en otros lenguajes de programación la sangría en el código es solo para legibilidad, la sangría en Python es muy importante.

Python usa sangría para indicar un bloque de código. 


                    

                        if 
                        5 
                        > 
                        2
                        :
                    

                    

                            print
                        (
                        "Five is greater than two!")
                        )

Python le dará un error si omite la sangría y obtendra un error de sintaxis: 


                    

                        if 
                        5 
                        > 
                        2
                        :
                    

                    

                        print
                        (
                        "Five is greater than two!")
                        )

El número de espacios depende de usted como programador, pero debe ser al menos uno. El número más usado son cuatro espacios en blanco de sangría.

Crear un comentario

Los comentarios comienzan con el carácter de numeral #, y Python los ignorará: 


                    

                        #Esto es un comentario
                    

                    

                        print
                        (
                        "
                        Hello, World!
                        "
                        )

Salida: 


                    

                        Hello, World!

Introducción

¿Qué es Python?

Python es un lenguaje de programación creado por Guido van Rossum a finales de los ochenta cuya filosofía hace hincapié en la legibilidad de su código. Su nombre está inspirado por la afición de su creador en el grupo de cómicos ingleses Monty Python.

Se trata de un lenguaje interpretado, miltiplataforma, con tipado dinámico, fuertemente tipado y orientado a objetos.

Es usado para diferentes propósitos

  • Desarrollo web (del lado del servidor)
  • Inteligencia artificial
  • Ciencia de datos
  • Desarrollo de software
  • Crear videojuegos
  • Scripting

¿Qué puede hacer Python?

  • Python se puede utilizar en un servidor para crear aplicaciones web
  • Python se puede utilizar junto con el software para crear flujos de trabajo
  • Python puede conectarse a sistemas de bases de datos. También puede leer y modificar archivos
  • Python se puede utilizar para manejar grandes cantidades de datos y realizar matemáticas complejas
  • Python se puede utilizar para la creación rápida de prototipos o para el desarrollo de software listo para producción

¿Por qué Python?

  • Python es un lenguaje multiplataforma, lo que significa que funciona en diferentes plataformas (Windows, Mac, Linux, Raspberry Pi, etc.)
  • Python tiene una sintaxis simple similar al idioma inglés
  • Python tiene una sintaxis que permite a los desarrolladores escribir programas con menos líneas que otros lenguajes de programación
  • Python se ejecuta en un sistema de interpretación, lo que significa que el código se puede ejecutar tan pronto como se escribe. Esto significa que la creación de prototipos puede ser muy rápida
  • Python puede tratarse de forma procedimental, orientada a objetos o funcional.
  • Python tiene una comunidad que incluye un grupo de personas increíblemente diverso y acogedor. Tener una comunidad bien conectada y de apoyo es fundamental para ayudar a resolver problemas, y la comunidad de Python apoya plenamente a personas que están aprendiendo Python como su primer lenguaje de programación.

Bueno saber

  • La versión principal más reciente de Python es Python 3, que se usará en esta documentación. Sin embargo, Python 2, aunque no se actualiza con nada más que actualizaciones de seguridad, sigue siendo bastante popular.
  • En esta documentación, Python se escribirá en la terminal y en un editor de texto. Puede utilizar un editor de código como Visual Studio Code, Sublime Text, Atom, o también en un entorno de desarrollo integrado, como Thonny, Pycharm, Netbeans, Eclipse, que son particularmente útiles cuando se administran colecciones más grandes de archivos Python. Además, también puede utilizar la terminal de Python o incluso el editor de texto si lo desea.

Sintaxis de Python en comparación con otros lenguajes de programación

  • Python fue diseñado para facilitar la lectura y tiene algunas similitudes con el idioma inglés con influencia de las matemáticas.
  • Python usa nuevas líneas para completar un comando, a diferencia de otros lenguajes de programación que a menudo usan punto y coma o paréntesis.
  • Python se basa en la sangría, usando espacios en blanco, para definir el alcance; como el alcance de los bucles, funciones y clases. Otros lenguajes de programación a menudo usan corchetes para este propósito.

Python es un gran lenguaje para aprender, ¡así que comencemos!

Empezar

Primero, verificaremos si hay una versión reciente de Python instalada en su sistema y la instalara si no lo está.

Python en Windows

Windows no siempre viene con Python, por lo que probablemente necesitara instalarlo.

Python en macOS

Python ya está instalado en la mayoría de los sistemas macOS, pero lo más probable es que sea una versión desactualizada de la que no querrá aprender.

Python en Linux

Los sistemas Linux están diseñados para la programación, por lo que Python ya está instalado en la mayoría de las computadoras Linux.

Instalar Python

Primero, verifique si Python está instalado en su sistema. Abra una ventana de terminal en Windows ingresando command (o terminal) en el menú Inicio y presione enter en la aplicación que aparece o presione la combinación de teclas Windows + r, y en la ventana que aparece escriba cmd y presione enter; en Mac abra una ventana de terminal yendo a Applications > Utilities > Terminal, también puede presionar la tecla Command, y escribir terminal y luego presionar Intro; en Linux abra una ventana de terminal ejecutando la aplicación Terminal en su sistema (en Ubuntu, puede presionar ctrl-alt-T).

En la ventana de terminal, ingrese python en minúsculas en Windows y Mac (en Linux escriba python3). Si obtiene un indicador de Python (>>>) como respuesta, Python está instalado en su sistema. Si ve un mensaje de error que indica que Python no es un comando reconocido, Python no está instalado.

En ese caso, o si ve una versión de Python anterior a Python 3, debe descargar un instalador de Python desde la página oficial. Vaya a la página oficial de Python y coloque el cursor sobre Downloads. Debería ver un botón para descargar la última versión de Python. Haz clic en el botón, que debería comenzar a descargar automáticamente el instalador correcto para su sistema.

Una vez que haya descargado el archivo, ejecute el instalador. Asegúrese de seleccionar la opción Add Python to PATH, lo que facilitará la configuración correcta de su sistema, y presione en Install now.

Iniciar con Python (Hola, Mundo!)

Una vez instalado Python en su sistema, abra la terminal de Python y escriba su primer programa. Ingrese la siguiente línea en su sesión de Python y asegúrese de ver el resultado: Hello, World! 

>>> print ("Hello, World!")
Hello, World!
>>>

Perfecto, ha instado Python correctamente en su sistema.

Sangría de Python

La sangría se refiere a los espacios en blanco al comienzo de una línea de código.

Mientras que en otros lenguajes de programación la sangría en el código es solo para legibilidad, la sangría en Python es muy importante.

Python usa sangría para indicar un bloque de código. 

if 5 > 2:
    print("Five is greater than two!"))

Python le dará un error si omite la sangría y obtendra un error de sintaxis: 

if 5 > 2:
print("Five is greater than two!"))

El número de espacios depende de usted como programador, pero debe ser al menos uno. El número más usado son cuatro espacios en blanco de sangría.

Variables

Las variables se utilizan para almacenar valores y se crean cuando le asigna un valor: 

x = 5
y = "Hello!"

Las variables no necesitan declararse con ningún tipo en particular, e incluso pueden cambiar de tipo después de que se hayan establecido. 

x = 4 # x es de tipo entero (int)
x = "Sally" # x ahora es una cadena (string)

Conversión de tipo de datos

Si desea especificar el tipo de datos de una variable, puede hacerlo con conversión: 

x = str(3) # x será '3'
y = int(3) # y será 3
z = float(3) # z será 3.0

Python distingue entre mayúsculas y minúsculas (Case sensitive)

Los nombres de las variables distinguen entre mayúsculas y minúsculas.

Esto creará dos variables: 

# A no es igual que a
a = 4
A = "Sally"

Nombrar y usar variables

Cuando usa variables en Python, debe cumplir con algunas reglas y pautas. Romper algunas de estas reglas provocará errores; otras pautas solo lo ayudan a escribir código que sea más fácil de leer y comprender. Asegúrese de tener en cuenta las siguientes reglas de variables:

  • Los nombres de las variables solo pueden contener letras, números y guiones bajos. Pueden comenzar con una letra o un guion bajo, pero no con un número. Por ejemplo, puede llamar a una variable message_1 pero no 1_message.
  • No se permiten espacios en los nombres de las variables, pero se pueden usar guiones bajos para separar palabras en los nombres de las variables. Por ejemplo, greeting_message funciona, pero greeting message provocará errores.
  • Evite el uso de palabras clave de Python y nombres de funciones como nombres de variables; es decir, no use palabras que Python haya reservado para un propósito programático particular, como la palabra print (Consulte en w3schools.com las palabras clave y funciones integradas de Python).
  • Los nombres de las variables deben ser breves pero descriptivos. Por ejemplo, name es mejor que n, student_name es mejor que s_n y name_length es mejor que length_of_persons_name.
  • Tenga cuidado al usar la letra minúscula l y la letra mayúscula O porque podrían confundirse con los números 1 y 0.

Constantes

Una constante es como una variable cuyo valor permanece igual durante la vida de un programa. Python no tiene tipos de constantes incorporados, pero los programadores de Python usan todas las letras mayúsculas para indicar que una variable debe tratarse como una constante y nunca cambiarse: 

MAX_CONNECTIONS = 5000

Cuando desee tratar una variable como una constante en su código, haga que el nombre de la variable sea todo en mayúsculas.

Números en Python (int y float)

Los números se utilizan con bastante frecuencia en la programación para llevar la puntuación en los juegos, representar datos en visualizaciones, almacenar información en aplicaciones web, etc. Python trata los números de varias formas diferentes, según cómo se utilicen. Veamos primero cómo Python gestiona los números enteros (o int), porque son los más sencillos de trabajar.

Enteros (int)

Puede sumar (+), restar (-), multiplicar (*), dividir (/) y elevar (**) enteros en Python. Además, es posible realizar otras operaciones con % (operador de modulo) y // (división de piso). 

>>> 2 + 3
5
>>> 3 - 2
1
>>> 2 * 3
6
>>> 3 / 2
1.5
>>> 3 ** 2
9
>>> 9 % 2
1
>>> 10 // 3
3

Numero de punto flotante (Float)

Python llama flotante a cualquier número con un punto decimal. Este término se utiliza en la mayoría de los lenguajes de programación y se refiere al hecho de que un punto decimal puede aparecer en cualquier posición de un número. 

>>> 0.1 + 0.1
0.2
>>> 0.2 + 0.2
0.4
>>> 2 * 0.1
0.2
>>> 2 * 0.2
0.4

Lista

Una lista es una colección de elementos en un orden particular. Puede acceder a cualquier elemento de una lista indicándole a Python la posición, o índice, del elemento deseado.

Ten en cuenta que Python considera que el primer elemento de una lista está en la posición 0, no en la posición 1.

Crear una lista

bikes = ['trek', 'redline', 'giant']
print(bikes)

Salida: 

['trek', 'redline', 'giant']

Obtener el primer elemento de una lista

bikes = ['trek', 'redline', 'giant']
first_bike = bikes[0]
print(first_bike)

Salida: 

trek

Obtener el último elemento de una lista

Python tiene una sintaxis especial para acceder al último elemento de una lista. Al solicitar el elemento en el índice -1, Python siempre devuelve el último elemento de la lista: 

bikes = ['trek', 'redline', 'giant']
last_bike = bikes[-1]
print(last_bike)

Salida: 

giant

(Antes de continuar se recomienda leer sobre el bucle for para entender cómo funciona el resto del código que se describe a continuación. Si no hablamos sobre el bucle for ahora es porque queremos llevar una secuencia ordenada sobre cada tema)

Recorrer una lista

bikes = ['trek', 'redline', 'giant']
for bike in bikes:

squares = []

Lista de comprensiones

El enfoque descrito anteriormente para generar los cuadrados de la lista consistió en utilizar tres o cuatro líneas de código. Una lista de comprensión le permite generar esta misma lista en una sola línea de código. Una lista de comprensión combina el bucle for y la creación de nuevos elementos en una línea, y agrega automáticamente cada nuevo elemento a la lista.

Sintaxis

newlist = [expression for item in iterable if condition == True]

Donde la condición es como un filtro que solo acepta los artículos que se valoran como True. El iterable puede ser cualquier objeto iterable, como una cadena, lista, tupla, conjunto, etc. Y la expresión es el elemento actual en la iteración, pero también es el resultado, que puede manipular antes de que termine como un elemento de la lista en la nueva lista:

El siguiente ejemplo crea la misma lista de números cuadrados que vio anteriormente, pero usa una lista de comprensión: 

squares = [value ** 2 for value in range(1, 11)]
print(squares)

Salida: 

[1, 4, 9, 16, 25, 36, 49, 64, 81, 100]

bikes = ['trek', 'redline', 'giant']

Declaración If

Las declaraciones if se utilizan para probar condiciones particulares y responder de manera apropiada.

Pruebas condicionales

Igual

x == 42

No es igual

x != 42

Mayor que

x > 42

Mayor o igual que

x >= 42

Menor que

x < 42

Menor o igual que

x <= 42

Una simple prueba condicional if

Python se basa en la sangría (espacio en blanco al comienzo de una línea) para definir el alcance en el código. Por lo que el código sangrado en la declaración if es el código que se ejecuta si la prueba se evalúa como verdadero. 

if age >= 18:
    print("You can vote!")

El resultado del código anterior se evaluará como verdadero y ejecutara el bloque de código dentro de la declaración if si la edad que estamos comprobando es mayor o igual a 18, si la edad es menor a 18 la prueba se evalúa como False y el bloque de código no se ejecuta. Los dos puntos al final de una instrucción le dicen a Python que "haga lo siguiente" o ejecute la siguiente línea o bloque de código sangrado. En nuestro ejemplo, la línea de código solo se ejecutará si la condición se evalúa como verdadera.

Declaraciones If-elif-else

La palabra clave elif es la forma que tiene Python de decir "si las condiciones anteriores no eran verdaderas, prueba esta condición". Se pueden utilizar tantas declaraciones elif como se deseen.

La instrucción else le permite definir una acción o un conjunto de acciones que se ejecutan cuando falla la prueba condicional (cuando fallan las declaraciones if y/o elif).

if age < 4:
    ticket_price = 0
elif age < 18:
    ticket_price = 10
else:
    ticket_price = 15
Diccionario

Un diccionario en Python es una colección de pares clave-valor. Cada clave está conectada a un valor y puede usar una clave para acceder al valor asociado con esa clave. El valor de una clave puede ser un número, una cadena, una lista o incluso otro diccionario. De hecho, puede utilizar cualquier objeto que pueda crear en Python como valor en un diccionario.

En Python, un diccionario está envuelto entre llaves, {}, con una serie de pares clave-valor dentro de las llaves, como se muestra en el ejemplo: 

dog = {'name': 'jake', 'color': 'brown'}
print(dog)

Salida: 

{'name': 'jake', 'color': 'brown'}

Cada clave está conectada a su valor por dos puntos y los pares clave-valor individuales están separados por comas. Puede almacenar tantos pares clave-valor como desee en un diccionario.

Acceder a un valor

Para acceder a un valor se utiliza la clave. Primero se escribe el nombre del diccionario seguido de corchetes y se escribe el nombre de la calve entre comillas. 

dog = {'name': 'jake', 'color': 'brown'}
print(dog['name'])

Salida: 

jake

Agregar un nuevo par clave-valor

dog = {'name': 'jake', 'color': 'brown'}
dog['breed'] = 'puddle'
print(dog)

Salida: 

{'name': 'jake', 'color': 'brown', 'breed': 'puddle'}

Recorrer todos los pares clave-valor

fav_numbers = {'eric': 17, 'ellie': 4}
for name, number in fav_numbers.items():
    print(f"{name} loves {number}")

Salida 

eric loves 17
ellie loves 4

Recorriendo todas las claves

fav_numbers = {'eric': 17, 'ellie': 4}
for name in fav_numbers.keys():
    print(f"{name} loves a number")

Salida 

eric loves a number
ellie loves a number

Recorrer todos los valores

fav_numbers = {'eric': 17, 'ellie': 4}
for number in fav_numbers.values():
    print(f"{number} is a favorite")

Salida 

17 is a favorite
4 is a favorite
Blucle for

Un bucle for se utiliza para iterar (repetir un proceso) sobre una secuencia (una lista, una tupla, un diccionario, un conjunto o una cadena). 

fruits = ['apple', 'banana', 'cherry']
for x in fruits:
    print(x)

Salida 

apple
banana
cherry

El concepto de bucle es importante porque es una de las formas más comunes en que una computadora automatiza tareas repetitivas. Por ejemplo, en un ciclo simple como el anterior, Python lee inicialmente la primera línea del ciclo: 

for x in fruits:

Esta línea le dice a Python que recupere el primer valor de la lista de frutas y lo asocie con la variable x. Este primer valor es "apple". Python luego lee la siguiente línea: 

print(x)

Python imprime el valor actual de x, que sigue siendo 'apple'. Debido a que la lista contiene más valores, Python regresa a la primera línea del ciclo y repite el mismo proceso. En la segunda iteración del ciclo for, Python recupera el segundo valor de la lista que es “banana” y lo almacena en la variable x, luego imprime el valor de x, que es “banana” en esta iteración, y repite el mismo proceso de nuevo para el resto de los elementos de la lista hasta que no haya más elementos en la lista.

Entrada del usuario (input)

Sus programas pueden solicitar al usuario que ingrese algún valor, esto se realiza con la función input. Toda la entrada se almacena como una cadena.

Pedir un valor al usuario

name = input("What's your name? ")
print(f"Hello, {name}!")

Salida 

What's your name?

Al ingresar un dato 

What's your name? Allan

Salida con un dato ingresado 

Hello, Allan!

Solicitar entrada numérica

age = input("How old are you? ")
age = int(age)

Salida 

How old are you?

En el ejemplo anterior la segunda línea del código convierte el valor ingresado por el usuario a un tipo de dato int. Esto se hace porque, como se mencionó anteriormente, toda entrada a través de la función input se almacena como una cadena.

Bucle While

Un ciclo while repite un bloque de código siempre que una determinada condición sea verdadera.

Un bucle while simple

current_value = 1
while current_value <= 5:
    print(current_value)
    current_value += 1

Salida 

1
2
3
4
5

Dejar que el usuario elija cuándo salir

msg = ''
while msg != 'quit':
    msg = input("What's your message? ")
    print(msg)

El código anterior indica al usuario que ingrese un mensaje y muestra ese mensaje en pantalla. El código deja de ejecutarse cuando el usuario ingresa la palabra quit.

Funciones

Las funciones son bloques de código con nombre, diseñados para realizar un trabajo específico. La información que se pasa a una función se llama argumento y la información que recibe una función se llama parámetro.

Una función simple

1  def greet_user()
2      """Display a simple greeting."""
3      print("Hello!")

4  greet_user()

Este ejemplo muestra la estructura más simple de una función. La línea 1 usa la palabra clave def para informar a Python que estás definiendo una función. Esta es la definición de función (function definition), que le dice a Python el nombre de la función y, si corresponde, qué tipo de información necesita la función para hacer su trabajo. Los paréntesis contienen esa información. En este caso, el nombre de la función es greet_user() y no necesita información para hacer su trabajo, por lo que sus paréntesis están vacíos. (Aun así, los paréntesis son obligatorios). Finalmente, la definición termina en dos puntos.

Cualquier línea con sangría que siga a def_greet_user(): constituye el cuerpo de la función. El texto en la línea 2 es un comentario llamado docstring, que describe lo que hace la función. Las cadenas de documentos están encerradas entre comillas triples, que Python busca cuando genera documentación para las funciones en sus programas.

La línea print ("Hello!") en la línea 3 es la única línea de código real en el cuerpo de esta función, por lo que greet_user() solo tiene un trabajo: print ("Hello!").

Cuando desee utilizar esta función, la llama. Una llamada a la función le dice a Python que ejecute el código en la función. Para llamar a una función, escriba el nombre de la función, seguido de cualquier información necesaria entre paréntesis, como se muestra en la línea 4. Como no se necesita información aquí, llamar a nuestra función es tan simple como ingresar greet_user(). Como era de esperar, imprime: Hello!.

Salida 

Hello!

Pasar un argumento

def greet_user(username):
    """Display a personalized greeting."""
    print(f"Hello, {username}!")

greet_user('jesse')

Salida 

Hello, jesse!

Valores predeterminados para parámetros

def make_pizza(topping='bacon'):
    """Make a single-topping pizza."""
    print(f"Have a {topping} pizza!")

make_pizza()
make_pizza('pepperoni')

Salida 

Have a bacon pizza!
Have a pepperoni pizza!

Regresar un valor

def add_numbers(x, y):
    """Add two numbers and return the sum."""
    return x + y

sum = add_numbers(3, 5)
print(sum)

Salida 

8
